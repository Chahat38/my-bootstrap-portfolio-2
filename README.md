# 🌐 Chahat Portfolio (Basic)

A clean and responsive personal portfolio website built using **HTML, CSS, Bootstrap, and JavaScript**.  
This project showcases my skills, projects, and contact information in a simple and professional layout.

---

## 🚀 Features

- Responsive design (Mobile, Tablet, Desktop)
- Modern Hero Section with introduction
- About Me section
- Skills showcase using icons
- Projects section (Portfolio, Boutique, Clinic)
- Contact form
- Clean and minimal UI design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  
- Bootstrap Icons
